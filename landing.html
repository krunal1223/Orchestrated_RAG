Capabilities

Every Layer, Engineered for Accuracy

Six core techniques working in concert to deliver precise, source-grounded answers from your documents.

Hybrid Retrieval

Combines dense vector search (Qdrant) with sparse BM25 keyword matching for maximum recall on every query.

Contextual Enrichment

LLM-generated context per chunk improves retrieval accuracy by 35–49%, as validated by Anthropic's research.

Query Decomposition

Splits complex, multi-part questions into focused sub-queries, then fuses results with reciprocal rank fusion.

CRAG Gate

Intelligent sufficiency check — if documents fall short, Tavily web search kicks in as a seamless fallback.

Self-Critique

Post-generation hallucination detection compares the answer against context and auto-corrects when needed.

Real-Time Streaming

Token-by-token SSE delivery with live pipeline stage indicators — watch the answer assemble in real time.

Architecture

9-Stage Orchestrated Pipeline

Every query passes through a carefully orchestrated sequence of retrieval, ranking, and reasoning stages.

Stage 01

Query Decomposition

Split complex questions into focused sub-queries for targeted retrieval.

Stage 02

HyDE Expansion

Generate hypothetical answers to improve embedding similarity matching.

Stage 03

Hybrid Retrieval

Vector search (Qdrant) + BM25 keyword search running in parallel.

Stage 04

RAG-Fusion

Reciprocal rank fusion merges results with source diversity enforcement.

Stage 05

Cohere Rerank

Neural reranking selects the top 8 most relevant chunks.

Stage 06

Context Compression

Cross-encoder sentence-level filtering removes noise from context.

Stage 07

CRAG Gate

Sufficiency check — triggers Tavily web search if documents are inadequate.

Stage 08

LLM Generation

Streamed answer generation using NVIDIA Nemotron via OpenRouter.

Stage 09

Self-Critique

Hallucination detection compares output against source context.

Quickstart

Up and Running in 4 Steps

From zero to querying your documents in under 5 minutes.

01

Install Dependencies

Clone the repo and install Python packages.

pip install -r requirements.txt
02

Set API Keys

Create a .env file with your free-tier keys.

GROQ_API_KEY=...
COHERE_API_KEY=...
TAVILY_API_KEY=...
03

Ingest Documents

Drop PDFs, DOCX, MD, or TXT into docs/ and run:

python ingest.py
04

Launch

Start the web server and open the UI.

python server.py
# → localhost:8000

Start Querying Your Documents

Drop your files, ask questions, get accurate answers backed by source citations.
